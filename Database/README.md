#### 데이터 언어
* 데이터 정의 언어(DDL, Data Definition Language)
  * 관계형 데이터베이스 정의
  * 데이터베이스 고유의 특성 포함
  * CREATE, ALTER, DROP ...
* 데이터 조작 언어(DML, Data Manipulation Language)
  * 데이터베이스 검색, 등록, 삭제, 갱신 등
  * 검색 및 업데이트를 위해 사용
  * INSERT, UPDATE, DELETE ...
* 데이터 제어 언어(DCL, Data Control Language)
  * access 제어 (사용 권한)
  * GRANT, REVOKE ...
